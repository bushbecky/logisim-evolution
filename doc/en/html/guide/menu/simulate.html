Le menu Simulation

Simulation Enclanchée

Si cette case est cochée, les circuits visualisés seront "en direct", c'est-à-dire que les valeurs se propageant dans le circuit seront mises à jour à chaque poke ou modification du circuit. If checked, circuits viewed will be "live:" That is, the values propagating through the circuit will be updated with each poke or change to the circuit.

L'option de menu sera automatiquement décochée si The menu option will be automatically unchecked if circuit oscillation is detected.

Redémarer la simulation

Efface tout ce qui concerne l’état du circuit actuel, comme si vous veniez d’ouvrir à nouveau le fichier. Si vous visualisez l'état d'un sous-circuit, toute la hiérarchie est effacée. Clears everything about the current circuit's state, so that it is as if you have just opened the file again. If you are viewing a subcircuit's state, the entire hierarchy is cleared.

Simulation pas à pas

Avance la simulation d'un pas en avant. Par exemple, un signal peut finir par entrer dans une porte au cours d'une étape, mais la porte n'affichera pas de signal différent avant la prochaine étape de simulation. Pour aider à identifier les points du circuit global qui ont changé, tous les points dont les valeurs changent sont indiqués par un cercle bleu; si un sous-circuit contient des points qui ont été modifiés (ou ses sous-circuits, récursivement), alors il sera dessiné avec un bleu Advances the simulation one step forward. For example, a signal may end up entering a gate during one step, but the gate won't show a different signal until the next simulation step. To help identify which points in the overall circuit have changed, any points whose values change are indicated with a blue circle; if a subcircuit contains any points that have changed in it (or its subcircuits, recursively), then it will be drawn with a blue outline.

Aller dans la vue

Lorsque vous passez à l'état d'un sous-circuit via son menu local, le sous-menu Sortir vers l'état répertorie les circuits ci-dessus. l'état actuel du circuit visualisé. La sélection de l'un affiche le circuit correspondant. When you delve into a subcircuit's state via its pop-up menu, the Go Out To State submenu lists the circuits above the currently viewed circuit's state. Selecting one displays the corresponding circuit.

Sortir de la vue

Si vous vous êtes plongé dans l'état d'un sous-circuit et que vous l'avez retiré, ce sous-menu répertorie les sous-circuits situés sous le circuit actuel. La sélection de l'un des circuits affiche le circuit correspondant. If you have delved into a subcircuit's state and then moved back out, this submenu lists the subcircuits below the current circuit. Selecting one of the circuits displays the corresponding circuit.

Un seul tick

Étapes à suivre dans la simulation. Steps one tick forward into the simulation. This can be useful when you want to step the clocks manually, particularly when the clock is not in the same circuit that you are currently viewing.

Tic activé

Commence automatiquement à cocher l'horloge. Cela n'aura d'effet que si le circuit contient des appareils d'horloge (dans la bibliothèque de câblage). L'option est désactivée par défaut. Starts automatically ticking the clock. This will have an effect only if the circuit contains any clock devices (in the Wiring library). The option is disabled by default.

Fréquence des tics

Vous permet de sélectionner la fréquence des tics. Par exemple, 8 Hz signifie que les tics se produiront huit fois par seconde. Un tic est l'unité de mesure de base pour la vitesse des horloges. Allows you to select how often ticks occur. For example, 8 Hz means that ticks will occur eight times a second. A tick is the base unit of measurement for the speed of clocks.

Notez que la vitesse du cycle d'horloge sera plus lente que la vitesse d'activation: l'horloge la plus rapide possible aura un cycle à un tic et un cycle à un tic; une telle horloge aurait un taux de cycle haut / bas de 4 Hz si les graduations se produisent à 8 Hz. Note that the clock cycle speed will be slower than the tick speed: The fastest possible clock will have a one-tick up cycle and a one-tick down cycle; such a clock would have up/down cycle rate of 4 Hz if the ticks occur at 8 Hz.
