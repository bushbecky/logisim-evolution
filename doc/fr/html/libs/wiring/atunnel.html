Cela est utile lorsque vous devez connecter des points éloignés dans le circuit et qu'un réseau de câbles rendrait le circuit beaucoup plus moche. L'illustration ci-dessous montre comment cela fonctionne. </ P> <blockquote> <img src = "../../../ img-libs / tunnel-demo.png" width = "204" height = "86"> </ blockquote> <p> Ici, les trois tunnels ont le même label, <em> a </ em>, et donc les trois points auxquels les tunnels sont connectés. (Si l'un des tunnels était étiqueté comme autre chose, comme <em> b </ em>, alors il ferait partie d'un ensemble de tunnels différent.) Le tampon contrôlé émet une sortie flottante puisque son entrée inférieure est 0. Cela conduit normalement le fil provenant du tampon contrôlé à être bleu; mais ici, il est vert foncé parce que la sortie flottante combine à travers le tunnel avec le 0 de la broche en bas. Si l'entrée de contrôle dans le tampon passe à 1, le tampon contrôlé alimentera 1 dans le tunnel, ce qui se combinera avec 0 à partir de la broche en bas pour aboutir à une valeur d'erreur; on verrait alors des fils rouges traverser les trois tunnels. </ p> <h2> Pins </ h2> <p> Un tunnel n’a qu’une seule broche dont la largeur de bit correspond à l’attribut Bits de données du tunnel. Cette broche n'est ni une entrée ni une sortie & mdash; les tunnels correspondants sont simplement connectés de manière transparente. </ p> <h2> Attributs </ h2> <p> Lorsque le composant est sélectionné ou ajouté, Alt-0 à Alt-9 modifie son attribut <q> Data Bits </ q> et les touches fléchées modifient son attribut <q> Orientation </ q>. </ p > <dl> <dt> Orientation </ dt> <dd> La direction vers laquelle pointe le tunnel. </ dd> <dt> Bits de données </ dt> <dd> Le nombre de bits du tunnel. </ dd> <dt> Label </ dt> <dd> Le texte dans l’étiquette associé au tunnel. Ce tunnel est connecté à tous les autres tunnels avec exactement le même libellé. </ Dd> <dt> Police de libellé </ dt> <dd> La police avec laquelle rendre l'étiquette. </ dd> </ dl> <h2> Comportement de l'outil Poke </ h2> <p> Aucun. </ p> <h2> Comportement de l'outil de texte </ h2> <p> Permet de modifier l’étiquette associée au tunnel. </ p> <p> <a href="../index.html"> Retour à <em> Référence de la bibliothèque </ em> </a> </ p> </ body> </ html>
