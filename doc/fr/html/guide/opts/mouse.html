L'onglet Pointeur

Par défaut, lorsque vous cliquez sur la souris dans la zone de travail de Logisim, l’outil sélectionné dans le panneau d'exploration est utilisé. Si vous cliquez avec le bouton droit de la souris ou en maintenant la touche Ctrl enfoncée, un menu contextuel apparaîtra pour le composant sous le curseur de la souris.

Logisim vous permet de modifier ce comportement, ce qui vous évite d'avoir à accéder à la barre d'outils et/ou à l'explorateur constamment. (Cela peut aussi être pratique si vous êtes gaucher). Chaque combinaison d'un bouton de la souris et d'une touche de modification (n'importe quel sous-ensemble de shift, contrôle et alt) peut être associée à un autre outil. L'onglet pointeur vous permet de configurer ces mappages.

  • Sur le côté gauche se trouve un explorateur où vous pouvez choisir l'outil que vous souhaitez mapper.

  • Sur le côté supérieur droit se trouve un rectangle qui permet d'insérer une nouvelle combinaison simplement et en dessous se trouve une liste des combinaisons associée aux clics de souris. Par exemple, si vous souhaitez créer de nouveaux fils en déplaçant le curseur, vous devez d'abord sélectionner l'outil de câblage dans l'explorateur (sous la bibliothèque de base) et ensuite vous effectuez la combinaison souhaitée en positionnant le curseur de la souris dans le carré où il est dit "Clic avec une combinaison définie comme Outil de câblage" si cette combinaison est déjà utilisée, le mappage sera remplacé par le nouvel outil. À l'avenir, dans le mode dessins cette combinaison de souris correspondra à l'outil sélectionné dans la barre d'outils ou dans le volet de l'explorateur.

  • Notez qu'en mode édition toutes les combinaisons non répertoriées dans la liste utilisent simplement l'outil sélectionné au moment de l'action.

  • Vous trouverez au-dessous de la liste le bouton Supprimer, avec lequel vous pouvez supprimer le mappage sélectionné dans le tableau au-dessus du bouton.
