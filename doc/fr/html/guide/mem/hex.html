Hex editor

Logisim inclut un éditeur hexadécimal intégré pour visualiser et éditer le contenu de la mémoire. Pour y accéder, ouvrez un menu contextuel pour le composant de mémoire et sélectionnez Modifier le contenu .... Pour les composants de la ROM, dont le contenu de la mémoire fait partie de la valeur d'attribut, vous pouvez également accéder à l'éditeur hexadécimal en cliquant sur le bouton correspondant. valeur d'attribut. Logisim includes an integrated hex editor for viewing and editing the contents of memory. To access it, bring up a pop-up menu for the memory component and select Edit Contents.... For ROM components, which have the memory contents as part of the attribute value, you can alternatively access the hex editor by clicking the corresponding attribute value.

Les chiffres en italique à gauche affichent les adresses mémoire écrites en hexadécimal. Les autres numéros affichent des valeurs à partir de cette adresse mémoire; l'éditeur hexadécimal peut afficher quatre, huit ou seize valeurs par ligne, en fonction de ce qui rentre dans la fenêtre. Pour aider à compter, chaque groupe de quatre valeurs a un espace plus grand entre. The numbers in italics at left display memory addresses, written in hexadecimal. The other numbers display values starting from that memory address; the hex editor may display four, eight, or sixteen values per line, depending on what fits in the window. To help with counting, each group of four values has a larger space between.

Vous pouvez naviguer dans la mémoire à l'aide de la barre de défilement ou du clavier (les touches fléchées, accueil, fin, page précédente et page suivante). La saisie de caractères hexadécimaux modifiera la valeur actuellement sélectionnée. You can navigate through memory using the scroll bar or using the keyboard (the arrow keys, home, end, page up, and page down). Typing hexadecimal characters will alter the currently selected value.

Vous pouvez sélectionner une plage de valeurs en faisant glisser la souris, en effectuant un clic-droit sur la souris ou en naviguant dans la mémoire avec le clavier tout en appuyant sur la touche Maj. Les valeurs peuvent être copiées et collées à l'aide du menu Edition. le presse-papiers peut également être transféré dans d'autres applications. You can select a range of values by dragging the mouse, shift-clicking the mouse, or navigating through memory with the keyboard while depressing the shift key. Values may be copied and pasted using the Edit menu; the clipboard can also be transferred into other applications.
