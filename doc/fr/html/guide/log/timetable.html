Fenêtre Chronogramme

Cette fenêtre permet la visualisation des signaux sélectionnés précédemment. Vous remarquerez le menu, la barre d'outils, deux zones d'affichage et les boutons.

Le menu windows ne contient qu'une option pour préciser si la fenêtre reste affichée au-dessus des autres ou si elle peut passer en arrière-plan. Par défaut elle reste au-dessus.

la barre de menu dispose de cinq boutons

 Permets d'activer la simulation ou de la stopper. Équivalente du menu | Simulation | | Simulation enclenchée |
 Permets d'avancer d'un tic dans la simulation. Équivalente du menu | Simulation | | un seul tic |
 Active ou désactive l'horloge sysclk. Équivalente du menu | Simulation | | tics activés |
 Avance d'une demi-période l'horloge sysclk. Équivalente du menu | Simulation | | 'Clk' un pas |
Avance d'une période ou d'une demi-période l'horloge clk. Équivalente du menu | Simulation | | Clk un seul tic |

Les boutons: On trouve trois boutons au fond de notre fenêtre.

  • Chager fichier :Charge et affiche le résultat depuis un fichier préalablement sauvegardé. Cela ne permet pas de paramétrer la simulation.

  • Exporter... :Exporte les données dans un fichier text. Vous devrez préciser le nom du fichier dans la fenêtre de sauvegarde.

  • Exporter image :Exporte le chronogramme dans une image de type png. Vous devrez préciser le nom du fichier dans la fenêtre de sauvegarde.
