Fenêtre chronogramme

Pour comprendre ou débugger un circuit il est souvent très utile de pouvoir observer les différents signaux de manière visuels. C'est le but du chronogramme. Ce module permet l'enregistrement des signaux sous forme graphique ou dans un tableau de valeur dans un fichier text.

Vous pouvez entrer dans le module de journalisation via le menu | Simulation | → | Chronogramme |. Il fait apparaître la fenêtre de sélection des signaux.

Le circuit ci-dessous est un exemple d'illustration pour le module chronogramme.

Ce sont les horloges qui servent de pilote à l'affichage des signaux. La simulation connait deux horloges particulières. Une est impérative et nommée sysclk et une autre optionnelle et secondaire nommée clk

Note: il est impératif qu'une horloge nommée sysclk apparaisse dans votre circuit. Elle sera utilisée comme base de temps par le module chronogramme. Il n'est pas nécessaire qu'elle soit connectée à votre circuit. Elle est en principe la plus rapide et est réglée sur un rapport cyclique 1/1 tic.

Remarque: les modules de journalisation sont en phase alpha (V2.14). Il peut être buggé et il est sujet à des changements importants dans le futur. Les rapports de bogues et les suggestions sont les bienvenus pour tout Logisim. Ils sont particulièrement bienvenus concernant ces nouvelles fonctionnalités. Si vous n'envoyez pas de commentaires, ils ne changeront probablement pas.
