The Selection tab

L'onglet Sélection vous permet de sélectionner les valeurs à inclure dans le journal. La fenêtre ci-dessous correspond au circuit suivant. The Selection tab allows you to select which values should be included in the log. The window below corresponds to the following circuit.



L'onglet est divisé en trois zones verticales. Le premier (le plus à gauche) est une liste de tous les composants du circuit dont les valeurs peuvent être consignées. Parmi les bibliothèques intégrées, les types de composants suivants prennent en charge la journalisation. The tab is divided into three vertical areas. The first (leftmost) is a list of all components in the circuit whose values can be logged. Among the built-in libraries, the following types of components support logging.

Bibliothèque de câblage: composants de broche, sonde et horloge
Bibliothèque d'E / S: composants de bouton et de LED
Bibliothèque de mémoire: tous les composants sauf la ROM Wiring library: Pin, Probe, and Clock components
I/O library: Button and LED components
Memory library: All components except ROM
Pour les composants auxquels des étiquettes sont associées, leurs noms correspondent aux étiquettes; les noms des autres composants spécifient leur type et leur emplacement dans le circuit. Tous les sous-circuits apparaîtront également dans la liste; ils ne peuvent pas être sélectionnés pour la journalisation, mais les composants éligibles peuvent l'être. Notez que le composant RAM nécessite que vous choisissiez les adresses mémoire à consigner; il permet de se connecter uniquement pour les 256 premières adresses. For components which have labels associated with them, their names correspond to the labels; other components' names specify their type and their location within the circuit. Any subcircuits will also appear in the list; they cannot be selected for logging, but eligible components within them can be. Note that the RAM component requires you to choose which memory address(es) should be logged; it allows logging only for the first 256 addresses.

La dernière zone verticale (la plus à droite) répertorie les composants sélectionnés. En outre, il indique la base (base) dans laquelle les valeurs multi-bits du composant seront enregistrées; la base n'a pas d'effet significatif sur les valeurs à un bit. The last (rightmost) vertical area lists those components that have been selected. Also, it indicates the radix (base) in which the component's multi-bit values will be logged; the radix does not have a significant effect on one-bit values.

La colonne centrale des boutons permet la manipulation des éléments dans la sélection. The middle column of buttons allows the manipulation of the items within the selection.

  • Addajoute les éléments actuellement sélectionnés du côté gauche dans la sélection. adds the currently selected item(s) on the left side into the selection.
  • Change Radixcycle la base du composant actuellement sélectionné dans la sélection entre 2 (binaire), 10 (décimal) et 16 (hexadécimal). cycles the radix for the currently selected component in the selection between 2 (binary), 10 (decimal), and 16 (hexadecimal).
  • Move Updéplace le composant actuellement sélectionné dans la sélection vers l'avant. moves the currently selected component in the selection forward one spot.
  • Move Downdéplace le composant actuellement sélectionné dans la sélection en arrière d'un point. moves the currently selected component in the selection back one spot.
  • Removesupprime le composant actuellement sélectionné dans la sélection. removes the currently selected component in the selection.
