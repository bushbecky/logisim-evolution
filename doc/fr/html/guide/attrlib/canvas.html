La surface de travail (Canvas)

La surface de travail est vraiment la partie la plus visible de Liogisim-evolution c'est la que vous dessinerez vos circuits et leur symbole pour enfint en simuler le fonctionnement.

Elle n'est composé que de quelques élements:

Les barres de défilement horizontal et vertical : Elles fonctionnent comme dans la plus part des programme il suffit de déplacer les curseurs pour déplacer la vue sur d'autre partie du dessins. Elles peuvent aussi se manipuler depuis le clavier ou la souris avec les combinaison suivante :

  • Mollette souris ou Fléche bas/haut : Défilement vertical
  • Alt-Mollette souris ou Fléche droite/gauche: Défilement horizontal
  • Touche Home : Haute de page
  • Touche End : Bas de page

Le zoom : En bas sur la gauche il y a l'affichage du facteur de zoom courant (100% dans la figure). vous pouvez le modifier avec les deux bouton a droite du taux ou avec Ctrl + Molette souris

Le bouton grille : A droite du zoom, Bascule du mode grille affichée au mode grille invisible alternativement.
