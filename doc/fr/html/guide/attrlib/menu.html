Les menus

Cette section explique rapidement les six menus qui accompagnent chaque fenêtre majeure de Logisim.

Le menu Fichier :Ce menu permet d'ouvrir de fermer de sauvegarder les projets. Il donne accès aux préférences et permet de quitter le programme.

Le menu Editer : Principalement utilisé pour modifier, copier, supprimer, dupliquer, déplacer des éléments de dessins.

Le menu Projet : Permet la gestion des librairies, de modifier l'ordre dans le panneau de navigation et modifier l'apparence de ce panneau. Il donne accès aux préférences de projet.

Le menu Simulation : Contrôle la simulation, lancer, stopper, passer en mode pas à pas, lancer les horloges.

Le menu Fenêtre :Réduire, Agarandir, fermer les fenêtre. Atteindre la fenêtre Analyse combinatoire (si activée) et les préférences d'application

Le menu Fenêtre :Réduire, Agarandir, fermer les fenêtres. Atteindre la fenêtre Analyse combinatoire (si activée) et les préférences d'application.

Le menu Aide :Allez dans le tutoriel, le guide de référence ou la référence des librairies.
