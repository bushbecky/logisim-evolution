Etape 3: Ajouter du texte

Il n'est pas nécessaire d'ajouter du texte dans votre circuit pour le faire fonctionner, mais si vous voulez présenter votre circuit à quelqu'un (comme un enseignant), alors quelques étiquettes permettront de mieux présenter le rôle des différents éléments de votre circuit.

Sélectionnez l'outil texte (). Vous pouvez cliquer sur une broche d'entrée et commencer à taper pour lui assigner un label. (Il est préférable de cliquer directement sur le symbole de l'entrée plutôt que de cliquer à l'endroit où vous voulez placer le texte, car de cette manière l'étiquette sera liée avec le symbole). Vous pouvez faire de même avec la broche de sortie. Ou vous pouvez juste cliquer sur n'importe quel emplacement et commencer à taper du texte pour mettre du text n'importe où ailleurs.

Beaucoup de composants acceptent des labels, par exemple si vous cliquez dans une des portes vous pouvez aussi lui donner un label

Il est possible de modifier un label de plusieurs autres manières.

    Avec un double-clic sur le composant avec l'outil d'édition().
    Avec un clic sur un label avec l'outil text().
    En modifiant la propriété label dans la fenêtre de propriété.
