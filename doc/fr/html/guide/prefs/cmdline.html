Command-line options

Vous pouvez configurer de nombreuses préférences d'application de Logisim via les options de ligne de commande. Cela peut être particulièrement utile dans un laboratoire d'ordinateurs composés d'étudiants célibataires où vous souhaitez que Logisim démarre la même chose pour les étudiants à chaque fois, indépendamment de la configuration du programme par les étudiants précédents. You can configure many of Logisim's application preferences via command line options. This can be particularly useful in a laboratory of single-student computers where you want Logisim to start up the same for students every time, regardless of how previous students may have configured the program.

The overall command-line syntax is as follows.

 
java -jar jarFileName [options] [filenames]

Les fichiers supplémentaires facultatifs nommés sur la ligne de commande seront ouverts en tant que fenêtres distinctes dans Logisim. The optional additional files named on the command line will be opened as separate windows within Logisim.

L'exemple suivant démarre Logisim dans sa configuration de base. The following example starts Logisim in its basic configuration. 

java -jar jarFileName -plain -gates shaped -locale en

Les options prises en charge sont les suivantes. Supported options include the following.

-plain
-empty
-template templateFile

Configure le modèle à utiliser par Logisim. Configures the template for Logisim to use.

-gates [shaped|rectangular]

Configure le type de porte à utiliser. Configures which type of gate to use.

-locale localeIdentifier

Configure la traduction à utiliser. Au moment d'écrire ces lignes, les paramètres régionaux pris en charge incluent: Configures which translation to use. As of this writing, the supported locales include:

deGerman
elGreek
enEnglish
esSpanish
frFrench
ruRussian
ruRussian
-accents [yes|no]

Cela ne concerne que les langues qui utilisent des caractères en dehors du jeu de caractères ASCII 7 bits; cela inclurait les langues utilisant des caractères accentués et n'inclurait pas l'anglais. Si no , les caractères en dehors du jeu de caractères ASCII 7 bits sont remplacés par des équivalents appropriés au langage; Cela serait utile pour les combinaisons Java / OS où ces caractères ne sont pas bien pris en charge. This is only relevant for languages that use characters outside the 7-bit ASCII character set; this would include languages using accented characters, and it would not include English. If no, characters outside the 7-bit ASCII character set are replaced with equivalents appropriate to the language; this would be useful for Java/OS combinations where such characters are not supported well.

-clearprops

Effacer toutes les préférences de l'application au démarrage, Logisim agira comme s'il était exécuté sur le système hôte pour la première fois. Clear all application preferences at startup, so Logisim will act as if it were being executed on the host system for the first time.

-nosplash

Masque l'écran de démarrage Logisim initial. Hides the initial Logisim splash screen.

-help

Affiche un résumé des options de la ligne de commande. Displays a summary of the command line options.

-version

Affiche le numéro de version de Logisim. Displays the Logisim version number.
