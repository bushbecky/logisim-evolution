Command-line options

Vous pouvez configurer de nombreuses préférences d'application de Logisim via les options de ligne de commande. Cela peut être particulièrement utile dans un laboratoire d'ordinateurs composés poste d'étude où vous souhaitez que Logisim démarre la même chose pour les étudiants à chaque fois, indépendamment de la configuration du programme fait par l'étudiant précédent.

La syntaxe globale de la ligne de commande est la suivante.

java -jar jarFileName [options] [filenames]

Les fichiers supplémentaires facultatifs nommés sur la ligne de commande seront ouverts en tant que fenêtres distinctes dans Logisim.

L'exemple suivant démarre Logisim dans sa configuration de base. 

java -jar jarFileName -plain -gates shaped -locale fr

Les options prises en charge sont les suivantes. 

-plain
-empty
-template templateFile

Configure le modèle à utiliser par Logisim.
Utilise modèle standard, modèle vide, modèle utilisateur

-gates [shaped|rectangular]

Configure le style de symbole de porte à utiliser.

-locale localeIdentifier

Configure la traduction à utiliser. Au moment d'écrire ces lignes, les paramètres régionaux pris en charge incluent:
de Allemand
el Grec
en Anglais
es Espagnol
fr Français
pt Portugais
ru Russe

-accents [yes|no]

Cela ne concerne que les langues qui utilisent des caractères en dehors du jeu de caractères ASCII 7 bits; cela inclurait les langues utilisant des caractères accentués et n'inclurait pas l'anglais. Si no , les caractères en dehors du jeu de caractères ASCII 7 bits sont remplacés par des équivalents appropriés au langage; Cela serait utile pour les combinaisons Java / OS où ces caractères ne sont pas bien pris en charge.

-clearprops

Effacer toutes les préférences de l'application au démarrage, Logisim agira comme s'il était exécuté sur le système hôte pour la première fois.

-nosplash

Masque l'écran de démarrage Logisim initial.

-help

Affiche un résumé des options de la ligne de commande.

-version

Affiche le numéro de version de Logisim.

-noupdates

Ne met pas à jour le logiciel.

-analyze

Affiche les menus analyse combinatoire.

-load file

load image file into RAM (works with -tty only)

-sub file1 file2

charger le fichier en remplaçant le fichier de bibliothèque1 par le fichier2

-tty format

exécuter sans interface graphique
