Le menu contextuel et les fichiers

Le menu contextuel

Le menu contextuel des composants mémoires comprend quatre options de plus que celles communes à tous les composants:

Editer le contenu :
Afficher un éditeur hexadécimal pour éditer le contenu de la mémoire.
Effacer le contenu :
Réinitialise toutes les valeurs en mémoire à zéro.
Charger l'image ... :
Réinitialise toutes les valeurs en mémoire en fonction des valeurs trouvées dans un fichier. Le format du fichier est reconu automatiquement et si ce n'est pas le cas une fenêtre s'ouvre pour une sélection appropriée.
Sauvegarder l'image ... :
Stocke toutes les valeurs en mémoire dans un fichier en utilisant un des formats décrit ci-dessous.

Les formats de fichier

  • v2.0 raw Version historique de logisim permet un codage "run.length"
  • v3.0 hex words plain Proche de la version v2.0 structuré en mot
  • v3.0 hex words addressed Comme précédement avec la possibilité de définir une adresse de départ pour chaque ligne du fichier.
  • v3.0 hex bytes plain big-endian
    v3.0 hex bytes plain little-endian Organisé en flux de byte.
  • v3.0 hex bytes addressed big-endian
    v3.0 hex bytes addressed little-endian Organisé en flux de byte comme précédement avec la possibilité de définir une adresse.
  • Binary data big-endian
    Binary data little-endian Image de la mémoire non codée en headecimale.
  • Ascii byte escape big-endian
    Ascii byte escape little-endian Fichier codé ASCII avec code escape come en C pour les caratères non imprimable.
